Anatomía de un elemento HTML: Atributos, anidamiento y elementos vacíos

{
    <h1> -> Etiqueta de apertura. Tiene que contener el inicio del elemento.
        Hola -> Contenido
    </h1> -> Etiqueta de cierre.
} -> Elemento.  

{ 
    <h1 
        class (atributo -> Van siempre en la etiqueta de apertura)
        ="saludo" (valor)
    > 

    class: Se le puede dar un identificador al elemento para poder modificarlo con CSS.

} -> Etiqueta de apertura.

Anidamiento -> Un elemento que contiene a otro. Tenerlo en cuenta a la hora de construir el HTML para 
su correcto renderizado.

Elementos vacíos -> Elementos que no necesitan un contenido, pero sí atributos especiales para poder
renderizarse.

    <img src="ejemplo.png" alt="Mascota">
